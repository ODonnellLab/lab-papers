# ODonnell Lab Papers — Claude Instructions

This repo is a curated reading list for the ODonnell Lab. The entire list lives in `README.md`.

## Structure

Papers are organized into sections by topic (e.g. `## C. elegans & the Microbiome`). Each entry follows this format:

```
- **[Title](URL)** — Authors. *Journal*, Year.
```

Use `### Background & Reviews` and `### Research Papers` subsections where a section is large enough to warrant it.

## Adding papers

When the user shares a URL or asks to add a paper:
1. Fetch the URL to get the title, authors, journal, and year.
2. Decide which existing section fits best. If none fits, create a new `##` section.
3. Append the entry to that section in `README.md`.
4. Commit and push.

## Editing

All edits go to `README.md`. Commit messages should be short: e.g. `add Baker et al. 2026 to GPCR section`.
